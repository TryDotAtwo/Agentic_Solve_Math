# User Prompts Log

## Назначение

Этот файл хранит пользовательские постановки задач максимально близко к оригиналу. Он является частью исследовательской и научной трассируемости workspace.

## Правила ведения

- Сохранять существенные пользовательские запросы дословно или почти дословно.
- Не сжимать формулировки до коротких summary, если исходные детали важны.
- Для каждого запроса фиксировать:
  - дату;
  - язык;
  - полный текст;
  - извлечённые требования;
  - связь с созданными документами и действиями.
