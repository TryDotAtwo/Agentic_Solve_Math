# Organization Tree

Эта папка хранит каноническую организационную структуру root-level multi-agent system и шаблонов подпроектных команд.

Здесь лежат:

- `root_command/` - executive role root orchestrator;
- `root_departments/` - департаменты корневой команды;
- `subproject_template_departments/` - шаблон департаментов для автономных подпроектов.

На текущем этапе это design scaffold для будущей кодогенерации и enforcement.
