# Subproject Template Departments

Здесь хранится шаблон оргструктуры автономной команды подпроекта.

Каждый реальный подпроект должен со временем получить у себя аналогичную локальную структуру, но уже внутри собственной папки и по собственным локальным правилам.

Базовый template target:

- 1 `Subproject Commander`
- 8 departments
- в каждом департаменте: 1 head + 3 staff agents

Итого: 33 agents на подпроектную команду.
