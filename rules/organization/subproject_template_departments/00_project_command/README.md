# Project Command

Executive role:
- `Subproject Commander`

Mission:
- принять handoff от root;
- запустить heads локальных департаментов;
- вести локальные цели и приоритеты;
- общаться с root по результатам и эскалациям.

Direct communication:
- с heads локальных департаментов
- с root boundary only through explicit runtime contracts
