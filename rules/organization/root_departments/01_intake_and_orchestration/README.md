# Intake and Orchestration

Head:
- `Head of Intake and Orchestration`

Staff:
- `Intake Analyst`
- `Context Packager`

Mission:
- разобрать intake;
- выделить основные файлы и контекст;
- подготовить task packets для других департаментов;
- инициировать запуск нужных команд.

Communication:
- вверх: только к `Root Orchestrator`
- в стороны: к heads других департаментов
- вниз: к своим staff

Writable scope:
- root-owned intake and run-prep artifacts
- не глобальные hard rules
