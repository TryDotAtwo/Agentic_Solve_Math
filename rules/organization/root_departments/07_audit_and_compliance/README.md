# Audit and Compliance

Head:
- `Head of Audit and Compliance`

Staff:
- `Access Auditor`
- `Result Verifier`

Mission:
- проверять корректность выводов;
- проверять соблюдение ACL и rule boundaries;
- разбирать спорные случаи и сомнения;
- быть независимой точкой проверки.

Communication:
- heads других департаментов могут обращаться напрямую
- staff по умолчанию через своего head

Writable scope:
- audit reports
- compliance notes
