# Root Departments

Здесь живут департаменты корневой команды.

Каждый департамент должен со временем иметь:

- `README.md`
- локальные правила и prompt contracts
- access policy
- role matrix
- manifest templates

Текущая структура рассчитана на 8 департаментов по схеме:

- 1 head
- 2 staff agents

Итого вместе с `Root Orchestrator` получается 25 root agents.
