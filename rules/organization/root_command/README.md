# Root Command

Эта папка описывает `Root Orchestrator`.

Его роль:

- принять intake и high-level user objective;
- запустить департаменты root team;
- общаться только с heads департаментов;
- утверждать критичные изменения контекста, целей, hard rules и restructuring decisions.

Обычная операционная работа должна спускаться вниз по иерархии, а не выполняться им напрямую.
